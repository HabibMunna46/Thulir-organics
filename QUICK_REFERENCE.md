## 🔧 Customization Quick Tips

### Change Colors
Edit in `css/style.css`:
```css
:root {
    --primary-color: #2E7D32;
    --secondary-color: #4CAF50;
    --accent-color: #8BC34A;
}
```

### Update Contact Info
Find in all HTML files (footer):
```html
<a href="tel:+919XXXXXXXXXX">+91 9XXXXXXXXXX</a>
<a href="mailto:email@domain.com">email@domain.com</a>
```

### Change Logo Icon
In navbar (all pages):
```html
<i class="fas fa-leaf"></i> <!-- Change 'leaf' to any FA icon -->
```

### Add Newsletter Email
In `js/script.js`, update WhatsApp number:
```javascript
const whatsappUrl = `https://wa.me/919176069176?text=...`;
```
