# Thulir Organics - Premium Organic Farming Website

## Overview

**Thulir Organics** is a production-ready, enterprise-grade website for an organic farming company specializing in premium agricultural products. The website features a modern, elegant, and professional design inspired by leading companies like Apple, Stripe, and Notion.

## 📝 Customization Guide

### Changing Colors
Edit CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #2E7D32;
    --secondary-color: #4CAF50;
    --accent-color: #8BC34A;
}
```

### Updating Content
1. Edit HTML files directly
2. Update company information in footer
3. Modify product details in `products.html`
4. Replace images with your own

### Adding New Pages
1. Create new HTML file
2. Copy header/footer from existing page
3. Create new CSS section if needed
4. Add navigation link

## 🐛 Troubleshooting

### Images Not Loading
- Check image URLs in HTML
- Ensure Unsplash/Pexels links are accessible
- Use relative paths for local images

### Forms Not Working
- Verify WhatsApp phone numbers
- Check browser console for errors
- Ensure JavaScript is enabled

### Styling Issues
- Clear browser cache
- Check CSS file paths
- Verify Bootstrap CDN is loaded
